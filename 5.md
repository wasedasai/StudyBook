## 第一節 コンポーネント指向
まずはコンポーネント指向について学びます。コンポーネントとは部品のことです。

例えばフッターはサイトの大半のページにおいて共通です。そこでフッターを部品としてあらかじめ定義しておきます。
```js
const Footer = () => {
  return (
    <footer>
      <div class="footer-inner">
        <h2>早稲田祭2021</h2>
        <ul class="footerNav">
          <li>Page1</li>
          <li>Page2</li>
          <li>Page3</li>
          <li>Page4</li>
        </ul>
      </div>
    </footer>
  );
}
```
